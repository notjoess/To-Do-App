# To-Do App (Minimalist)

This package includes a **backend** (NestJS + SQLite) and a **frontend** (React) for a minimalist To-Do List.

## Backend
- Location: `backend`
- Runs on: `http://localhost:3001`
- Swagger API docs: `http://localhost:3001/api`

### Setup (backend)
1. cd backend
2. npm install
3. npm run start:dev
(You need Node.js and npm installed.)

## Frontend
- Location: `frontend`
- Runs on: `http://localhost:3000`

### Setup (frontend)
1. cd frontend
2. npm install
3. npm start

## Notes
- Database file `tasks.db` will be created automatically in the backend folder.
- If ports conflict, change frontend proxy or backend port.
